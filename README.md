# adaptive-wellness-planner

An interactive wellness planning tool that helps organize nutrition, training, recovery, and daily routines into a practical decision-support system.

## Overview

This project consolidates personal wellness and planning frameworks into a single interactive interface.

## Features

- Adaptive daily planning
- Multiple daily pathways based on energy levels
- Seasonal wellness recommendations
- Nutrition and training guidance
- Interactive navigation between modules
- Structured decision-support framework

## Technologies
- HTML
- CSS
- AI-assisted development

## Purpose
The project explores how personal wellness knowledge can be translated into an accessible interactive system that supports everyday decision-making and long-term habit sustainability.
