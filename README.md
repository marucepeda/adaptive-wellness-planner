# adaptive-wellness-planner

An interactive web application designed to transform complex personal knowledge into a practical decision-support system.

## Overview

This project was created to transform scattered knowledge, routines, and planning strategies into a single interactive tool.

It combines factors such as:

- Energy levels
- Seasonal changes
- Nutrition
- Exercise
- Daily planning
- Habit management

## Features
- Interactive navigation
- Context-based recommendations
- Seasonal adaptations
- Planning and routine support

## Technologies
- HTML
- CSS
- JavaScript
- AI-assisted development

## Purpose
The goal of this project is to explore how complex information can be structured into a usable and accessible system that supports everyday decision-making.
